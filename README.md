# Лабораторна робота №3

## Тема
Git та Linux CLI

## Модель
Математичне моделювання перехідних процесів в електричному ланцюзі (RLC)

## Автор
Борейчук Б.М., група AI-235

## Опис
Реалізація обчислювальної моделі з курсу «Обчислювальні методи». 
Програма моделює динаміку струму та напруги в RLC-контурі за допомогою системи диференціальних рівнянь. 
Код написаний на Python.

## Як запустити
1. Переконайтеся, що встановлено Python та бібліотеку scipy (`pip install scipy`).
2. Запустіть скрипт командою:
   python main.py

## Залежності
- Python 3.x
- SciPy
